Selected Tabs Bookmark Manager

How to install in Brave:
1. Extract this folder somewhere permanent on your computer.
2. Open brave://extensions
3. Turn on Developer mode.
4. Click Load unpacked.
5. Select this folder.
6. Pin the extension from the extensions toolbar if you want quick access.

How to use:
1. In a Brave window, Ctrl/Cmd-click tabs to highlight multiple tabs.
2. Click the extension icon.
3. Choose Tabs to save:
   - Highlighted tabs
   - Active tab
   - All tabs in current window
4. Choose Destination:
   - Existing folder, then optionally Overwrite
   - New folder, then enter a name
5. Click Save bookmarks.

Notes:
- Internal pages like brave:// and some extension pages are skipped.
- If no highlighted tabs are found, the extension falls back to the active tab.
- Overwrite deletes existing bookmarks in the selected folder before saving.
- Recursive overwrite also deletes subfolders inside the selected folder.

Updating:
- Replace files in the folder.
- Then go to brave://extensions and click the reload icon on the extension card.
