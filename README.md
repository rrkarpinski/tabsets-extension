# TabSets Sync
TabSets Sync is a lightweight Brave/Chrome extension that treats bookmark folders as persistent tab sets.
It compares your current set of tabs with a saved set folder, shows the diff, and lets you either merge new tabs into the saved set or sync the saved set so it matches the current set exactly.

Like tab groups but without sessions. Utilise the existing bookmark browser support and their persistence, portability and serialisability to track tab sets that serve as workspaces/projects/collections.
